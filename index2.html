Question-1: What are the different ways to select an element in the DOM?

We can use document.getElementByID,document.querySelector,document.querySelectorAll, document.getElementsByClassname,document.getElementByTagName. Through these we store the element in a variable and process further operations.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML takes the whole element including the tag, any kind of text (visible or invisible) inside the tag.
innerText gives the visible texts only without the tags.
textContent gives only the texts(visible and invisible both).

Question-3: What is event delegation in the DOM?

It is a technique to reduce time and space complexity by writing only a single code and make it propagate to the parent node from the target node

Question-4: What is event bubbling in the DOM?

Event Bubbling refers to the mechanism of an event to trigger from the target element and propagate or bubble up to the root element.

Question-5: How do you create, add, and remove elements using JavaScript?

To create element we use document.createELement(TagNAme)
To add element we use document.appendChild(variable). [Here we store the element(that was created earlier) in the variable
To remove an element first we select the element with specific selector and the use .remove() method.
